# Just-JD Network Scanning Tool

A Python-based Network Health and Security Assessment Toolkit.

## Features

* Ping Test
* DNS Resolution
* Local IP Detection
* Default Gateway Detection
* Traceroute Analysis
* Packet Loss Test
* ARP Device Discovery
* Subnet Analysis
* Port Scanner
* Network Health Score

## Technologies Used

* Python
* Ping3
* Socket
* Scapy
* Subprocess
* ipaddress
